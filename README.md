## 🖥️ Projeto

Projeto Web Responsivo capsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias

Projeto desenvolvido durante o NLW da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
